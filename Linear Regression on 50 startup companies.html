📝 Multiple Linear Regression with Python¶

🗃️ Data Set¶

We are going to use the 50_CompList dataset,which stores some infomation about 50 start-up companies The data contains the following columns:

R&D Spend = Money spend for Reasearch and Development Administration Spend = Money spend for Administartion Marketing Spend = Money spend for marketing State = In which state company started Profit = How much profit gain
